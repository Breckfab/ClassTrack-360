# ClassTrack 360

Sistema de gestión pedagógica para profesores.

## Stack
- Python + Streamlit
- Supabase (base de datos)
- GitHub (control de versiones)

## Instalación local

1. Clonar el repositorio
2. Instalar dependencias:
   pip install -r requirements.txt
3. Ejecutar:
   streamlit run classtrack360.py

## Funcionalidades
- Agenda con historial de clases y sensor de tarea pendiente
- Gestión de alumnos e inscripciones
- Notas y calificaciones
- Gestión de cursos
